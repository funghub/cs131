cpg ()

This function copies a file into a specified directory and goes to that directory. 
It works by having two conditions in which the first if condition tests whether 
the second variable ($2) entered is a directory (-d). 
If that is the case, it will copy the file entered as the first variable ($1) to that directory ($2). 
Then that first condtion will follow up by changing the current directory to that specified in $2. 
Otherwise, the second else condition will only copy the file entered as the first variable ($1) 
to the file entered as the second variable ($2).

gofile () {

This is a shell function to change the current directory to a path specified by the user 
as the first argument.
